### Required Tags (Team 1)
✅ `<header>` - Organization name and logo  
✅ `<nav>` - Navigation links  
✅ `<main>` - Main content area  
✅ `<section>` - 4 sections (Mission, Programs, Donate, Impact)  
✅ `<article>` - 4 articles inside sections  
✅ `<aside>` - Quick facts sidebar  
✅ `<footer>` - Contact info and copyright  
✅ `<h1>` to `<h6>` - All heading levels used
